#古尔马尔格(Gulmarg)经济型酒店

由于古尔马尔格和克什米尔是印度的一部分，您可以找到非常实惠的住宿。只要付您以往在西方世界一家简易酒店房费的三分之一价钱，在这里您就可能预订到非常舒服的酒店，甚至有点豪华。

重要的是要知道不管在哪儿，都是一分钱一分货的，古尔马尔格也不例外。

##舒适和豪华程度：简介

如果您决定入住古尔马尔格我们所归为“经济型”的酒店，您将会遇到以下情况。

+ 酒店有中央供暖，意味着所有客房均由建筑内的中央空调供暖。
+ 大多数房间即使在室外冰冷时也很舒适。如果运气不好，你可能会分到一个风扇有点吵、玻璃窗有缝，盖着塑料片，有一丝冷风灌入的房间。
+ 走廊可能没有供暖，但休息室和餐馆在忙季都有暖气。在这里，所有人都外出滑雪时通常会停止供暖，节省能源。
+ 浴室很干净。所有浴室均配有间歇泉（热水壶），为客人提供热水淋浴。
+ 所有酒店都提供餐饮服务，有餐厅，入住管饭。
+ 服务还算专业，大多数员工都可以用英语沟通。
